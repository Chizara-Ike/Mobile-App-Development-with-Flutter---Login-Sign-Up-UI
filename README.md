# Mobile-App-Development-with-Flutter---Login-Sign-Up-UI
A clean and responsive Flutter authentication UI featuring Login and Sign Up screens, built to demonstrate modern mobile app design principles and Flutter best practices.
This project focuses on user interface design, form inputs, and reusable widgets, and serves as a foundation for integrating backend authentication in future iterations.

**Tech Stack:**

Flutter

Dart

Material Design

**Features:**

Login screen UI

Sign up screen UI

Reusable input and button widgets

Responsive layout for different screen sizes
